Задание на 19-03-18

Разработать и реализовать архитектуру приложения «товары в магазинах», 
включающую не менее 4-х классов.

Предусмотреть использование:
- readonly
- const
- перегрузка методов
- инкапсуляция
- несколько конструкторов
- this
- именованные аргументы
- необязательные аргументы

Не использовать: 
- наследование
