Лабораторная работа №1
Основы языка программирования C# и работа с массивами

Объявить одномерный (5 элементов ) массив с именем A и двумерный массив (3 строки, 4 столбца) дробных чисел с именем B. Заполнить одномерный массив А числами, введенными с клавиатуры пользователем, а двумерный массив В случайными числами с помощью циклов. Вывести на экран значения массивов: массива А в одну строку, массива В — в виде матрицы. 
Найти в данных массивах:
- общий максимальный элемент
- минимальный элемент
- общую сумму всех элементов
- общее произведение всех элементов
- сумму четных элементов массива А
- сумму нечетных столбцов массива В.

Даны 2 массива размерности M и N соответственно. Необходимо переписать в третий массив одинаковые значения первых двух массивов без повторений. 

Дан двумерный массив размерностью 5x5, заполненный случайными числами из диапазона от –100 до 100. Определить сумму элементов массива, расположенных между минимальным и максимальным элементами.

Предусмотреть использование:
- Констант (const)
- Явного и не явного приведения типов
- Форматированного вывода в консоль
- Отдельных методов класса Array
- Передачу по ссылке и по значению
